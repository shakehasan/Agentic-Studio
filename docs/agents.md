# Agents

## Supervisor / Campaign Director

Classify intent, decompose work, choose routing modes, enforce gates, and synthesize campaign packages.

## Market Research Agent

Build personas, demand signals, market narratives, segmentation, and audience assumptions.

## Competitive Intelligence Agent

Map alternatives, positioning, differentiation, SWOT, pricing narratives, and comparison frames.

## Content Strategist Agent

Create funnel architecture, content pillars, channel mix, editorial calendar, and messaging hierarchy.

## Copywriter Agent

Produce headlines, landing-page copy, ad variants, calls to action, and narrative copy systems.

## SEO / GEO Agent

Cluster keywords, map search intent, design on-page recommendations, internal links, and answer-engine citation plans.

## Social Media Agent

Create channel-specific posts, thread structures, publishing cadence, hooks, and ready-to-publish specs.

## Email Marketing Agent

Build lifecycle sequences, subject-line variants, segmentation logic, and send-cadence plans.

## Creative Brief Agent

Write visual briefs, storyboards, image-generation prompts, and brand-aligned art direction artifacts.

## Brand Voice & QA Agent

Verify tone, claims, consistency, compliance risks, and return pass or revise verdicts with targeted corrections.

## Analytics & Optimization Agent

Define KPI frameworks, measurement plans, experiments, projected impact, and optimization roadmap.
